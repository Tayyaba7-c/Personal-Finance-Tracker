# Personal Finance Tracker with Expense Prediction (Prototype)

## Project Overview
[cite_start]This is a functional subset of the Personal Finance Tracker, developed as part of the CS619 Prototyping Phase[cite: 1, 49]. [cite_start]The objective is to demonstrate development competence through a working web application[cite: 8].

## Implemented Features (Phase 1)
* [cite_start]**User Authentication**: A secure registration and login system[cite: 16, 58].
* [cite_start]**Expense Management**: Users can add and view expenses with Date, Amount, Category, and Description[cite: 20, 21, 61].
* [cite_start]**Database Integration**: Powered by SQLite for efficient data storage and CRUD operations[cite: 5, 30, 54].
* [cite_start]**Data Visualization**: Interactive Pie Chart displaying category-wise expense distribution using Matplotlib[cite: 6, 37, 66].

## Technical Stack
* [cite_start]**Backend**: Python (Flask Framework)[cite: 4, 75].
* [cite_start]**Frontend**: HTML, CSS, and Bootstrap[cite: 12, 75].
* [cite_start]**Database**: SQLite[cite: 30, 76].
* [cite_start]**Visualization**: Matplotlib[cite: 55, 77].

## How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install flask flask-sqlalchemy matplotlib`.
3. Run the application: `python app.py`.
4. Access via browser at `http://127.0.0.1:5000`.
